# Steam Show

Статичный сайт проекта **Steam Show**. Чистый HTML + CSS + vanilla JS, без фреймворков и сборки.

## Статус
Базовая заготовка (Hello World). Контент и дизайн добавляются по ходу работы.

## Локальный запуск
Сборки нет — достаточно открыть `index.html` в браузере. Для корректных относительных путей:

```bash
python -m http.server 8000
# http://localhost:8000/index.html
```

## Деплой
GitHub Pages с ветки `main` (root). После включения Pages сайт будет доступен по адресу
https://solardoll.github.io/steam-show/
