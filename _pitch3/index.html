Три разных концепта

Теперь это не один скелет с разными шапками, а три принципиально разные структуры: разная навигация по шоу, типографика и «фишка». Тёплый дух и твои ингредиенты — общие, собраны по-разному.

01 — Saga

Saga

Кинематографичная история-свиток. Антиква, дракон-spine рисуется по скроллу, фон разгорается от ночи к огню.

Шоу = «главы» с крупными медиа поочерёдно (+ ряд «всё сразу» вверху). Самый «дорогой/арт». Открыть → 02 — Reel

Reel

Киноплёнка. Anton, перфорация плёнки между секциями, hero с выделенным цветом словом.

Шоу = горизонтальная лента-катушка: тянешь вбок, кадр за кадром. Энергично, кинематографично. Открыть → 03 — Wall

Wall

Редакторская витрина. Модерн-гротеск, hero = живая фото-стена, много воздуха.

Шоу = интерактивный текст-индекс: список с превью-картинкой за курсором. Лучшее под «портфолио». Открыть →
