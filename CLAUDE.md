# CLAUDE.md

Память проекта для Claude Code. Основные правила и контекст описаны в @AGENTS.md — следуй им.

## Непреложные правила (из AGENTS.md)
1. **Язык общения — только русский.**
2. **Всегда коммить и пушь работу.** После каждого логически завершённого шага: stage → commit → `git push origin main`. Деплой с `main` на GitHub Pages автоматический.
3. **Стек только vanilla** — HTML + CSS + JS, без фреймворков и сборки.

## Репозиторий и деплой
- GitHub: `SolarDoll/steam-show`, рабочая ветка `main`.
- Live (после включения Pages): https://solardoll.github.io/steam-show/

## Что это за проект
Статичный сайт проекта **Steam Show**. Пока это базовая заготовка — конкретика по контенту,
секциям и дизайну не зафиксирована и добавляется по ходу работы.

## Как запустить / посмотреть локально
Сборки нет. Открыть `index.html` в браузере. Для проверки относительных путей удобнее поднять локальный сервер:
```bash
python -m http.server 8000
# затем открыть http://localhost:8000/index.html
```

## Окружение (Windows / PowerShell)
- `gh` после установки через winget может не подхватиться в активной сессии — обновить PATH или перезапустить шелл.
- В PowerShell не использовать `-f` у `gh api` (конфликт с оператором форматирования) — использовать `--field` / `--raw-field`.
- Для неинтерактивного ssh добавлять `-o StrictHostKeyChecking=accept-new`.
