# AI Agents Guidelines

Инструкции и правила, которым обязаны следовать AI-ассистенты (Antigravity, Claude и др.) при работе в этом репозитории.

## Core Rules
1. **Всегда коммить и пушить работу.** После каждого логически завершённого шага: stage изменений → commit с понятным сообщением → push в `origin`. Сайт деплоится с `main` на GitHub Pages автоматически.
2. **Язык общения — только русский.** Отвечай пользователю по-русски.
3. **Язык сайта — английский** (если не указано иное).

---

## Project Purpose & Context
Этот репозиторий содержит статичный сайт проекта **Steam Show**.

> ⚠️ Конкретика по контенту, структуре секций и визуальному дизайну пока НЕ зафиксирована.
> Это базовая заготовка — наполнение и финальные правила добавляются по мере работы.

## Tech Constraints
- Стек: чистый **HTML + CSS + vanilla JS**. Без фреймворков и систем сборки.
- Сайт должен оставаться лёгким, статичным и легко деплоиться (GitHub Pages).
- Удобство ручного редактирования важнее «умной» архитектуры.

## What NOT to Change Without Asking
1. Не добавлять JS-фреймворки или системы сборки — только vanilla HTML/CSS/JS.
2. Не делать финальных фиксаций дизайна (палитры, типографика) в `README.md` / `AGENTS.md` — эти доки про UX, структуру и поддержку.
